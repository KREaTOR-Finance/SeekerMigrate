## Important Behavioral Differences

> **Note:** The migration changes how authentication works. Please review these differences:

- Session persistence moves from Firebase servers to the device's wallet app
- User identity changes from email address to wallet public key
- Password recovery is replaced by wallet recovery phrase (handled by wallet app)
- Multi-device sync requires user to connect wallet on each device
